4. Les formulaires en PHP

ATELIER 1

Écrivez un script (nommé formulaire-atelier1.php) qui affiche le contenu du tableau nommé $_GET.
Il faut faire un echo "<pre>";var_dump($_POST);echo "<pre>";
Appelez, avec l'URL de votre navigateur, ce formulaire en lui passant 2 paramètres demo1.php?nom=marcel&age=20

ATELIER 2

Reprennez les indications sur support de cours (diapo 16) et construisez un fichier nommé formulaire-atelier2.html.
Le formulaire, quand il est soumis, doit transférer ses données en méthode GET vers le script formulaire-atelier2.php.
Le script formulaire-atelier2.php doit simplement afficher les données avec un var_dump.
Reprennez le traitement du script pour afficher "vous êtes mineur" si l'age est inférieur (ou égal) à 18 et "vous êtes un senior" si l'age est supérieur à 70.

ATELIER 3

Reprennez le formulaire de l'exercice précédent, mais vous allez maintenant travailler avec la méthode POST.

ATELIER 4

Écrivez un formulaire (nommé formulaire-atelier4.html) qui demande le nom, le prénom, le sexe et l'âge de l'utilisateur (M ou Mme).
Ce formulaire appelle (en méthode POST) un script formulaire-atelier4.php qui affichera
Bonjour monsieur Truc
ou
Bonjour madame Bidule
remplacez truc et bidule par le vrai prénom+nom de la personne, bien entendu !

ATELIER 5

Reprennez le travail de l'atelier 4, mais maintenant, la page doit aussi indiquer l'année de naissance de l'utilisateur.
Pour cela il faut connaitre l'année actuelle et soustraire l'âge.
La fonction date( paramètres ) du PHP permet d'obtenir l'année courante; paramètre va spécifier, dans la fonction, le format que l'on veut obtenir.
Il faut consulter la documentation php.net/manual/fr/function.date.php dans notre cas, la bonne expression est date( 'Y' )
